In diesem Auftrag geht es um erfolgreiches Ranking

  2. Erstellen Sie nun ein neues HTML & CSS Dokument

  3. Danach sollten sie folgende Datei herunterladen:
    Der Text, welcher in der Datei enthalten ist, muss in die HTML-Datei eingefügt werden.

  4. Danach soll dieser Text einigermassen sinnvoll formatiert werden, damit man die Übersicht beim Lesen nicht aus den Augen verlieren kann.
