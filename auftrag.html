Da Sie nun mehr über SEO wissen, können Sie mit den folgenden Aufträge Ihr Wissen testen.


In diesem Auftrag geht es um Media Queries



  1. Erstellen Sie zu erst eine HTML und CSS Dokument

  2. Schreiben Sie einen beliebigen Text (mind. 4 Sätze) und ändern Sie die Hintergrundfarbe. (Sie können auch ein Bild einfügen)

  3. Nun legen sie die maximale Breite des Textes auf 1000px fest.

  4. Fügen Sie im CSS ein "@media" ein, so wie es bei der Smartphone Optimierung gezeigt wird. Kleiner Tipp: @media (max-width) bedeutet, ab dieser Breite soll... und @media (min-width) bedeutet, bis zu dieser Breite, sonst...

  5. Versuchen Sie mithilfe des Tipps, dass bei einer Bildschirmbreite von 660px der Text auf eine Breite von 350px reduziert wird und sich die Hintergrundfarbe / Bild ändert.

  6. Schicken Sie uns Ihre Lösung, indem Sie uns die Datei per Email "fabio.zanon@bbz-sh.ch" oder "aleks.cvetanovic@bbz-sh.ch" schicken. Wir korrigieren Ihren Auftrag so schnell wie wir können! :)


Bei dieser Übung geht es um die Kundenfreundlichkeit



  1. Erstellen Sie eine Navigation mit einer ungeordneten Liste. (<ul>).

  2. Erstellen Sie 3 <li>-Elemente mit folgenden Begriffen: Kunden, Sitzung, Sonstiges

  3. Positionieren Sie diese Liste auf der rechten Seite und so, dass sie auf der oberen Kante des Fenster kleben bleibt (kleiner Tipp: Um dies prüfen zu können, kopieren Sie viel Text in das HTML-File hinein, sodass man runterscrollen kann und so die Positionierung der Liste prüfen kann.)

  4. Stellen Sie den Abstand zwischen diesen <li>-Elementen auf padding: 0 10px 0 10px; ein.

  5. Ändern Sie die Hintergrundfarbe der ganzen Navigation auf Grün.

  6. Verlinken sie die <li>-Elemente auf andere Webseiten. Dazu nutzt man das Element <a>.
