Erklärung

Was ist SEO?

Die englische Abkürzung SEO bedeutet "Search Engine Optimizer", auf Deutsch: Suchmaschinen Optimierung. Darunter versteht man ein Hervorheben der Webseite bezogen auf das Ranking, mit wenigen Hilfsmittel.


Indexierung

Ein gutes Beispiel dafür ist die Suchmaschine "Google". Jede Webseite wird gecrawlt (der Quellcode wird ausgelesen und an den Index gesendet). Diese Seite wird dann nach vielen verschiedenen Rakingfaktoren sortiert und eingereiht. Um eine Webseite indexieren zu können, muss man von einer anderen Webseite ein Link bekommen. Danach muss ein Crawler auf der fremden Webseite vorbeischauen und den Link zu unserer Seite entdecken. Diese Möglichkeit der Indexierung ist recht langwierig und unsicher.


OnPage

Mit OnPage meint man Ergänzungen oder Verbesserungen, die sich auf der Webseite selbst bzw. im Code befinden. D.h wie man seine Website mit dem Code optimiert. Folgende Punkte können dabei entscheidend sein:


Suchbegriff-Optimierung

Je mehr Wörter auf der Website mit dem Suchbegriff der User übereinstimmen, desto höher bzw. weiter vorne wird die Website geranked/platziert. Dies erfolt automatisch.

Keyword-Optimierung

Die Keywords sind der Schlüssel für ein hohes Ranking. Gerade auf den Titel der Website achtet Google besonders. Somit ist es wichtig, einen passenden Titel zu finden, der sich möglichst mit einem meist verwendetem Suchbegriff übereinstimmen könnte.









Überschriften-Optimierung

Die Überschriften soll IMMER die Kernaussage des Textes beinhalten. Das ist das, was die Website ausmacht und platziert. Der Suchmechanismus zälht das als Wichtigstes. Denn die Benutzer können sich mit den passenden Überschriften gut auf der Website zurechtfinden. Die Regel lautet: Der Inhalt steht über allem. SEO hat eine goldene Aussage: Content is King!






Smartphone-Optimierung

Damit die Handy-User die Website gleich sehen wie Computer User, muss die Website dafür extra optimiert werden. Falls man diesen Schritt auslässt könnte es sein, dass die Site zur Hälfte abgeschnitten angezeigt werden würde. Dies lässt sich mit Media Queries verhindern. Das Bild unten links zeigt die Situation in CSS. Nach @media und der Klammer, kommen die Selektoren. In diesem Fall kommt jetzt der Selektor "p", welches für den Text verantwortlich ist. Neben @media in der Klammer steht, ab welcher Breite der Site, die Eigenschaften in den Selektoren gültig sind.













OffPage


Bei OffPage geht es hauptsächlich um die externe Verlinkung seiner eigenen Website, damit diese mehr Aufmerksamkeit und somit mehr User bekommt. Verlinkungen von anderen Websites zu seiner eigenen nennt man "Backlinks". Dies ist das A und O bei der Kategorie OffPage. Doch wie bekommt man solche Backlinks? Sind sie gratis? Welche Gefahren lauern dahinter?


Wie bekommt man Backlinks?

Wenn man solche Backlinks bekommen möchte, muss man mit den Herstellern der Website Kontakt aufbauen. Selbstverständlich ist das erwerben von Backlinks nicht kostenlos. Firmenabhängig, gemessen an der Beliebtheit und Radius der Kundengrösse, können die Preise rasch nach oben gehen. Es gibt sehr viele Dienstleister, die sich auf das kommerzielle Geschäft mit Backlings (also deren Verkauff) spezialisiert haben. Gegen entsprechende Bezahlung wird versprochen in oft kürzester Zeit mehrere Hundert oder sogar Tausend Backlinks zu erzeugen.

Doch welche Gefahr steckt dahinter?

Die Nutzung solcher Angebote ist allerdins unbedingt abzuraten. Denn wenn eine Suchmaschine wie z. B. Google erkennt, dass für eine Website Backlinks im grossen Stil gekafut wurden, kann dies extrem negative Auswirkungen auf die Positionierung dieser Website haben. Deshalb gilt: Qualität vor Masse!

Technische SEO

Eine Suchmaschine lernt mit der Zeit, was für einen Inhalt die Website hat, wie oft er erneuert wird und ob Teile davon gelöscht werden. Darum spielt die Website-Indexierung eine wichtige Rolle. Die Website-Indexierung lässt sich für die Suchmaschinen deutlich erleichtern und die SEO entscheidend vorantreiben. Besonders wichtig ist die technische Webseite-Optimierung für Webseiten, welche dynamischen Content beinhalten. Schlussendlich muss die Suchmaschine diese Webseiten-Dynamik verstehen.



Webseite indexieren
  1. Unter http://www.google.de/addurl/ kann man einen Antrag auf Indexierung für eine Webseite stellen. Eine erfolgreiche Übermittlung der Daten ist aber keine Garantie auf eine Aufnahme. Kundenfreundlichkeit

Wenn man beim Coden der Website auf die Kundenfreundlichkeit achtet, ist die Chance höher, dass die Website Kunden anlockt, oder sie dazu führt, auf der Site zu bleiben. Häufig bei Websites, die als Shop dienen sollten, klicken die User weg, wenn es für sie unübersichtlich erscheint. Genau dies sollte man lieber schon beim Coden verhindern.
